
# Air Raids Map

Map of air raid alerts in regions of Ukraine


## API Reference

#### Get all alerts data

```http
  GET /api/alerts
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/PershynaMilana/air_raid_map.git
```

Go to the project directory

```bash
  cd air_raid_map
```

### Go to the backend directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```

### Go to the frontend directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
